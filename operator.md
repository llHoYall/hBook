# Operator

- **=** : Assignment
- **==** : equal to
- **<** : less than
- **<=** : less than or equal to
- **>** : greater than
- **and**, **&&** : logical AND
- **or**, **||** : logical OR
- **&** : bitwise AND
- **|** : bitwise OR

```ruby
>> true and false
=> false
>> true && false
=> false
>> true or false
=> true
>> true || false
=> true
```
